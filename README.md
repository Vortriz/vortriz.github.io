# Personal Portfolio Website

This is a personal portfolio website built with Astro.

## Project Setup

1.  **General dependencies**:

    ```bash
    nix develop
    ```

2.  **Install dependencies**:

    ```bash
    bun install
    ```

3.  **Run the development server**:

    ```bash
    bun run dev
    ```

4.  **Build for production**:
    ```bash
    bun run build
    ```
