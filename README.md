# Personal Portfolio Website

This is a personal portfolio website built with Astro.

## Project Setup

1.  **General dependencies**:

    ```bash
    nix develop
    ```

2.  **Install NPM dependencies**:

    ```bash
    npm install
    ```

3.  **Run the development server**:

    ```bash
    npm run dev
    ```

4.  **Build for production**:
    ```bash
    npm run build
    ```
